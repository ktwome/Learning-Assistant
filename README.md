# 학습 보조 어시스턴트

PDF 문서를 분석하여 사용자의 학습을 돕는 서비스입니다.

# 기능들

- PDF 파일 업로드 및 마크다운 파일 변환
- 사용자 질문에 따른 RAG 수행
- 난이도별 문제 출제

# 개발 히스토리
